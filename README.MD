# 臉部偵測-[開發中]

隨著 COVID-19 的確診人數不斷增加，越來越多單位會要求進入公司或校園前需要量體溫並戴口罩。本專案目的是為了降低人力成本，在單位入口架設普通鏡頭及紅外線鏡頭，自動化對穿過入口的人進行快速檢查。

## 流程
1. 使用 OpenCV Dnn 臉部偵測模型擷取畫面中的臉部
1. 將擷取到的臉部照片切成9宮格，取得額頭區塊，將此區塊溫度平均
1. 將擷取到的臉部透過訓練的模型偵測是否有戴口罩
